# Encuesta de satisfacción - GitHub Pages + Google Sheets

## Incluye
- formulario multipaso
- ficha de datos al final
- barra de progreso
- navegación siguiente / atrás
- campos de cargo y servicio contratado con sugerencias mientras se escribe
- integración con Google Apps Script y Google Sheets
