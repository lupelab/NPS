# Encuesta de satisfacción premium

Incluye:
- bloques por pantalla
- barra de progreso
- navegación adelante / atrás
- ficha de datos al final
- autocompletado en cargo y servicio contratado
- envío a Google Sheets vía Google Apps Script

## Archivos
- `index.html`
- `styles.css`
- `app.js`
- `Code.gs`

## Publicación en GitHub
Sube todo el contenido del proyecto a un repositorio y publícalo con GitHub Pages.

## Google Sheets
La pestaña debe llamarse `Respuestas`.
